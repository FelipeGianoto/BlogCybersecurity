    Protocolos de Rede

  • TCP/IP: O protocolo TCP/IP é o padrão da Internet e é amplamente utilizado para comunicação em redes locais e globais. É composto por dois protocolos principais, o TCP (Transmission Control Protocol) e o IP (Internet Protocol).

  • DNS: O protocolo DNS (Domain Name System) é usado para traduzir nomes de domínio em endereços IP.

  • DHCP: O protocolo DHCP (Dynamic Host Configuration Protocol) é usado para atribuir dinamicamente endereços IP a dispositivos em uma rede.

  • FTP: O protocolo FTP (File Transfer Protocol) é usado para transferir arquivos entre computadores em uma rede.

  • HTTP: O protocolo HTTP (Hypertext Transfer Protocol) é usado para transferir páginas da web e outros conteúdos na internet.

  • SMTP: O protocolo SMTP (Simple Mail Transfer Protocol) é usado para enviar e-mails.

  • SSH: O protocolo SSH (Secure Shell) é usado para acesso remoto seguro a sistemas e dispositivos em uma rede.

  • SSL/TLS: O protocolo SSL/TLS (Secure Sockets Layer/Transport Layer Security) é usado para criptografar a comunicação entre servidores e clientes em redes, como em transações bancárias online.
